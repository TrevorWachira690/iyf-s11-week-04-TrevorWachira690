## Project Description
I built an index.html file that contains javascript code in a separate file called main.js and a mini project which was a students grade management system written in javascript. This was built to teach myself about Javascript and how it is used in programming.

## Technologies Used
- HTML
- JavaScript
- VScode
- Git
- Github

## Features
- Index.html- This is the file which runs in the browser and contains Javascript code from main.js.
- main.js - This is the Js file which contains practice codes which run smoothly in the browser's console.
- grade-management.js- This is the Javascript mini project that manages the grades of each student.

## How to Run
1. Clone this repository
2. Open `index.html` in your browser
3. Open the console.

**OR**
1. Open the repository in VS code
2. Open the terminal
3. Write the code `node main.js` or `node grade-management.js`
4. Press enter
